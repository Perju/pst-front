Temporizadores

Solo hay que indicar un intervalo de tiempo entre envios del mensaje y el mensaje a enviar, el nombre sirve solo para identificarlo.

Comandos

Se componen de un nombre que sera el que se utilice en el chat precedido por una exclamacion (!), un tiempo de espera para que se pueda utilizar el comando, el nivel de usuario donde el 1 es el streamer y el 5 todos los espectadores y por ultimo el mensaje de respuesta al comando.

Como configurar los tokens

para configurar los token de la app, el "appToken" y el "appSecret" hay que ir a https://dev.twitch.tv/console con nuestra cuenta de streamer de twitch y crear una aplicación. Ahi solo tendremos que copiar el appToken y generar un Secret y guardarlo bien porque no se podra volver a ver.

Para el chat hay que ir a esta página https://twitchaddon.page.link/1Sk5 y crear un token para el chat con la cuenta que queramos usar como bot que habla en el chat.
